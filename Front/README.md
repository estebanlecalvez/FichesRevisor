

# Install Back Env
 `cd Back`
 `pip install pipenv`
 `pipenv install Django`
 `pipenv install djangorestframework`
 `pipenv install django-cors-headers`
 `pipenv install`
 `pipenv shell`
` pipenv update`

# Run server
 `python manage.py runserver`

# Migration
`python manage.py makemigrations`
`python manage.py migrate`

# Install Front
`cd Front`
`npm install`
`npm start`
